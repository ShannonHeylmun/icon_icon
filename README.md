# icon_icon

This is a [Flutter](flutter.dev) project built to showcase and help developers utilize a selection icons and glyphs. While this was developed with my own Flutter workflows in mind, it should be useful for anyone working with the glyph sets offered herein. Visit [the live web app here](https://shannonheylmun.github.io/icon_icon/).

Search for Emoji by name, category, ASCII, code point, and skin variants. Copy emoji (click/tap) or code point (double click/tap) to the system clipboard. Within the Apple ecosystem, this provides for pasting across networked devices (if using).
